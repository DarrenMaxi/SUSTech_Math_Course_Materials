Step-3

The determinant of a Permutation matrix is either plus one or minus one. In this case, if n is even then the determinant is –1 and if n is odd then the determinant is +1.

To understand this, let us see how we can obtain identity matrix from such a Permutation matrix.

First we interchange the last two rows. Therefore, the new matrix has the last row coincidental with the last row of the identity matrix. Then we interchange the second last and third last rows and so on! Thus, in an Permutation matrix, we have to carry out interchange of rows to obtain the identity matrix.
