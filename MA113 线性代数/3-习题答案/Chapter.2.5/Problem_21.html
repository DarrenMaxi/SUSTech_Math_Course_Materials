We follow the basic difference between an incident matrix and an adjacency matrix.

In an incidence matrix, the row heads are edges and column heads are nodes where as in an adjacency matrix, both the row as well as the column heads are nodes such that if there is an edge from ith node to jth node, then the entry in the ijth place is 1 and otherwise zero.
